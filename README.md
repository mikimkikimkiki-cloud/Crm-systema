# Образовательный Центр - Веб-сайт

Веб-сайт для образовательного центра по программированию, созданный на чистом HTML, CSS и JavaScript

## Возможности

### Для пользователей:
- ✅ Главная страница с информацией о центре
- ✅ Каталог курсов с описанием и ценами
- ✅ Расписание занятий с фильтрацией
- ✅ Система отзывов студентов
- ✅ Страница "О центре" с информацией о преподавателях
- ✅ Форма обратной связи
- ✅ Система регистрации и авторизации (LocalStorage)
- ✅ Личный кабинет пользователя
- ✅ Онлайн-чат для поддержки
- ✅ Адаптивный дизайн для мобильных устройств

### Для администратора:
- ✅ Админ-панель (CRM система)
- ✅ Управление студентами (добавление, редактирование, удаление)
- ✅ Управление преподавателями
- ✅ Управление курсами
- ✅ Управление оплатами и статусами платежей
- ✅ Управление отзывами
- ✅ Просмотр обращений пользователей
- ✅ Дашборд со статистикой

## Использование

1. Откройте `indexhtml` в браузере
2. Все данные сохраняются в LocalStorage браузера
3. Для начала работы зарегистрируйте нового пользователя

## Особенности

- **Хранение данных**: Все данные (пользователи, курсы, отзывы, сообщения) хранятся в LocalStorage
- **Без backend**: Полностью клиентское приложение, не требует сервера
- **Адаптивность**: Дизайн адаптирован для мобильных устройств
- **Интерактивность**: Онлайн-чат, формы обратной связи, система регистрации

## Функционал

### Регистрация и авторизация
- Регистрация новых пользователей
- Вход в систему
- Выход из системы
- Защита личного кабинета

### Курсы
- Просмотр всех доступных курсов
- Описание курсов, цены, длительность
- Запись на курс (требуется авторизация)

### Расписание
- Просмотр расписания занятий
- Фильтрация по курсам и дням недели

### Отзывы
- Просмотр отзывов студентов
- Добавление новых отзывов

### Личный кабинет
- Просмотр записанных курсов
- История платежей
- Статистика обучения

### Поддержка
- Онлайн-чат на всех страницах
- Форма обратной связи

### Админ-панель
- **Вход:** Логин: `лола`, Пароль: `123456`
- Управление всеми данными системы через удобный интерфейс
- Полный контроль над студентами, курсами, оплатами и отзывами
