# Инструкция по запуску

## Быстрый старт

1. Откройте файл `indexhtml` в любом современном браузере
2. Сайт готов к использованию!

## Функции

### Тестовый аккаунт
Вы можете зарегистрировать нового пользователя через страницу "Регистрация".

### Данные
Все данные хранятся в LocalStorage браузера, поэтому:
- Данные сохраняются между сессиями
- Данные уникальны для каждого браузера
- При очистке кэша браузера данные будут удалены

### Основные страницы
- **index.html** - Главная страница
- **courses.html** - Каталог курсов
- **schedule.html** - Расписание занятий
- **reviews.html** - Отзывы студентов
- **about.html** - О центре
- **contacts.html** - Контакты и форма обратной связи
- **login.html** - Вход в систему
- **register.html** - Регистрация
- **dashboard.html** - Личный кабинет (требует авторизации)

## Возможности

✅ Регистрация и авторизация пользователей
✅ Просмотр и запись на курсы
✅ Просмотр расписания с фильтрацией
✅ Просмотр и добавление отзывов
✅ Личный кабинет с записанными курсами
✅ Онлайн-чат поддержки (на всех страницах)
✅ Форма обратной связи
✅ Адаптивный дизайн для мобильных устройств

## Примечания

- Сайт работает полностью на клиентской стороне (без сервера)
- Для продакшена рекомендуется подключить backend-сервер
- Платежная интеграция не реализована (требуется подключение платежных систем)
- Для реального использования рекомендуется добавить шифрование паролей на сервере
